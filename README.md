# JasonSileo.github.io

This repository powers my undergraduate portfolio website using GitHub Pages. It includes the following assets:

- **Resume:** `assets/resume.pdf`
- **Profile photo:** `assets/profile.jpg`

## Editing the Site

To make changes, edit `index.html` directly. The site uses plain HTML and inline CSS for simplicity.

Add new projects or sections by modifying the appropriate `<section>` in the HTML. After committing and pushing, GitHub Pages will automatically rebuild the site.

## Deployment

The site is served from the `main` branch via GitHub Pages. To update the live site:

1. Make your changes locally.
2. Commit and push to `main`.
3. Visit `https://<your-github-username>.github.io/` in your browser to see the updates.
